# Full Monthly SEO Audit

## Required Inputs

- Current board/ticket state.
- Latest audit data register.
- URL registry.
- Latest source exports or explicit statement that none are newer.
- Prior monitoring queue.

## Required Output

- Prioritized current work batch.
- Stale finding cleanup.
- New or changed issue summary.
- Register updates.
- Monitoring close/reopen decisions.

## Evidence Rules

- State export date and path before analysis.
- Separate commercial targets from expected noise.
- Do not close indexing findings without recrawl/export evidence or an
  expected-excluded closure reason.
